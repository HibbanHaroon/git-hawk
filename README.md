# GitHawk

A fun project which retrieves GitHub information for your GitHub profile and some other cool things.

## Run Locally

Clone the project

```bash
  git clone https://github.com/HibbanHaroon/git-hawk.git
```

Go to the project directory

```bash
  cd git-hawk
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
